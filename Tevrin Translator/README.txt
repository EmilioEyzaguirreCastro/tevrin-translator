PARA USAR EL TRADUCTOR TIENES QUE INSTALAR OLLAMA
Ollama es el modelo de IA que usa el traductor de Tevrin. Este instala un modelo de IA en la computadora para usarlo con el traductor (no hay plata para la API de chatgpt)

1. Instale Ollama desde: https://ollama.com/
2. Una vez instalado, abra un cmd y ejecute el comando
	ollama run mistral
3. Una vez terminada la instalación de mistral, puede cerrar el cmd y ejecutar el programa

IMPORTANTE: Cada vez que inicie su PC, tendrá que iniciar Ollama antes de ejecutar Tevrin Translator, usando el mismo comando:
	ollama run mistral
